What is Git?

  • A distributed version control system
  • A toolkit for dealing with changes
  • A bunch of command line utilities

Why do I like git?

  • Staging is useful
  • Local branching is fast
  • It has that Unix feel
  • github is great

Lingo: Nouns

  • working copy
  • stage
  • commit
  • branch
  • repository
  • master branch
  • origin server

Lingo: Verbs

  • to stage
  • to commit
  • to branch
  • to push
  • to fetch
  • to merge

How I git started

  1. I install git on my machine
  2. I set my git configuration
  3. I create or clone a repository

Install git

  • Did you do it?
  • See http://help.github.com/

Configure git

  • Edit ~/.gitconfig
  • See http://gist.github.com/563331 for mine

Create a repository

git init

Clone an existing repository

git clone

How I use git daily

  1. I write code
  2. I stage chunks of changes
  3. I commit to "master"
  4. GOTO 1

Stage some changes

git add --patch git remove

Commit my changes

git commit

How I share code on github

  1. I fetch commits from github
  2. I push my commits to github
  3. I merge remote changes into master

Fetch changes from a remote repository

git fetch

Merge remote changes

git merge origin/master

Push commits to a remote repository

git push origin master

How I work on "big" features

  1. I create a branch
  2. I switch to the branch
  3. I code, stage, commit in the branch
  4. I switch back to "master" to do other work
  5. I merge the branch into "master"
  6. I might push a branch to github
  7. I might track a branch on github

Create a new branch

git checkout -b <branch>

Switch to a branch

git checkout <branch>

Merge one branch into another

git merge <branch>

Start a local copy of a shared branch

git branch --track <server> <branch>

How I deal with disaster

  • I reset my working copy
  • I amend the last commit
  • I undo the last commit
  • I resolve conflicts during merge
  • I fetch and merge before I push

Reset the working copy

git reset --hard HEAD

Undo the last commit

git reset --soft HEAD
